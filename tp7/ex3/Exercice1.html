Compétences techniques acquises : TP 7 - Introduction au JavaScript et Manipulation du DOM

  • Syntaxe de base JavaScript : Apprentissage des variables (let, const), des types de données et des structures de contrôle (boucles, conditions).
  • Sélection d'éléments : Maîtrise des méthodes de sélection d'éléments HTML via querySelector et getElementById.
  • Gestion des événements : Utilisation de addEventListener pour rendre la page réactive aux actions de l'utilisateur (clics, survol, saisie clavier).
  • Modification dynamique du contenu : Capacité à modifier le texte (textContent), le style (style.property) ou les classes CSS (classList) d'un élément en temps réel.
  • Logique algorithmique : Création de scripts permettant de résoudre des problèmes simples, comme des calculatrices, des compteurs ou des vérificateurs de formulaires.

Commentaire de synthèse

    « Le TP 7 a marqué un tournant majeur : le passage d'une page statique à une page intelligente et interactive. L'apprentissage du JavaScript m'a permis de comprendre comment le navigateur interprète le code pour modifier l'interface sans recharger la page. J'ai trouvé particulièrement stimulant de voir comment quelques lignes de script peuvent transformer radicalement l'expérience utilisateur. Cette première approche de la manipulation du DOM pose les fondations nécessaires pour construire des applications web modernes et dynamiques. »
