Compétences techniques acquises : TP 8 - JavaScript Avancé et API Fetch

  • API Fetch : Capacité à récupérer des données distantes depuis des fichiers JSON ou des serveurs externes.
  • Asynchronisme (Async/Await) : Maîtrise de la gestion des promesses pour exécuter des tâches en arrière-plan sans bloquer l'affichage.
  • Traitement des données JSON : Apprentissage du parsing et de l'extraction d'informations au sein d'objets JavaScript complexes.
  • Génération dynamique de contenu : Utilisation des boucles (forEach, map) pour injecter automatiquement des listes d'éléments dans le HTML.
  • Gestion des erreurs réseau : Mise en place de sécurités avec try...catch pour gérer les pannes de serveur ou les erreurs de chargement.

Commentaire de synthèse

« Le TP 8 a été une étape fondamentale pour comprendre comment rendre une page web réellement "vivante". L'apprentissage du Fetch m'a permis de connecter mes interfaces à des sources de données réelles, transformant ainsi mon projet en une véritable application capable de se mettre à jour sans rechargement. La gestion de l'asynchronisme a été un défi stimulant qui m'a appris à mieux structurer mon code pour garantir une expérience utilisateur fluide, même lors de temps de chargement longs. »
