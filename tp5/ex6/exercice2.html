L'Effet Parallaxe

C'est quoi le parallaxe ?

Le parallaxe est un effet visuel où l'image d'arrière-plan se déplace plus lentement que le contenu au premier plan lors du défilement (scroll), créant une illusion de profondeur.

Étape 1 : Le HTML

Créez simplement des blocs <section>. Certains contiendront vos images de fond, d'autres votre texte explicatif.

Étape 2 : Le secret du CSS

Le secret réside dans une seule ligne de code CSS : background-attachment: fixed;.

Cette propriété indique au navigateur que l'image de fond ne doit pas bouger par rapport à la fenêtre, même si on descend dans la page.

Magique, non ?

Compétences acquises :

  • Maîtrise de background-attachment: fixed pour l'immersion.
  • Utilisation de background-size: cover pour des images plein écran.
  • Gestion de la profondeur visuelle sans JavaScript.
